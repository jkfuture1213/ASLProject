# Smart Ledger
AI 기반 소비 분석 가계부

C#과 WinForms을 이용하여 제작한 스마트 가계부 프로그램입니다.  
사용자의 소비 내역을 기록하고, 소비 분석 기능을 통해 소비 패턴 및 절약 방향을 제공합니다.

---

# 프로젝트 소개

기존 가계부 프로그램은 단순 기록 기능에 집중되어 있습니다.  
본 프로젝트는 단순한 지출 기록을 넘어, 사용자의 소비 데이터를 분석하여 더 효율적인 소비 습관 형성을 돕는 것을 목표로 합니다.

본 프로그램은:

- 수입/지출 내역 관리
- 카테고리별 소비 분석
- 월별 소비 통계
- 소비 패턴 분석

기능을 제공합니다.

# 사용법
1. 회원가입 버튼을 누르면 뜨는 창에서 아이디와 비밀번호를 입력하고 계정을 만듭니다.
2. 아이디와 비밀번호를 입력해서 로그인 버튼을 누릅니다.
3. 활성화되는 지출 내역 창에서 날짜, 구분, 카테고리, 메모, 금액을 입력하고 추가를 누릅니다.
4. 지출 내역을 다 추가했으면 지출 내역 분석 버튼을 누르고 지출 분석 결과를 확인합니다.

# 주요 기능

## 가계부 기능
- 수입/지출 내역 추가
- 수입/지출 내역 수정 및 삭제
- 날짜별 내역 조회
- 카테고리 관리

## 데이터 관리
- 로컬 데이터 관리

## 통계 기능
- 월별 소비 통계
- 카테고리별 소비 비율 분석
- 소비 패턴 시각화

## AI 분석 기능
- 과소비 항목 탐지
- 소비 패턴 분석
- 절약 방향 제안
- 소비 습관 인사이트 제공
